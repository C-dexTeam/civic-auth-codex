# Codex - Civic Auth Integration

Codex is a Web3 education platform built on Solana.  
This repo demonstrates the integration of [Civic Auth](https://www.civic.com/) using the `@civic/auth-web3` SDK for secure user authentication and embedded wallet creation.

## ✨ Features

- Civic Auth integration
- Embedded wallet for each user
- Custom login/logout buttons
- User profile display after login

## 🚀 Getting Started

```bash
git clone https://github.com/C-dexTeam/civic-auth-codex.git
cd civic-auth-codex
npm install
npm run dev
